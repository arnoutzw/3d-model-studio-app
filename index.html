Firebase Database Setup

Setup instructions:

  1. Go to console.firebase.google.com
  2. Create a new project (or select existing)
  3. Go to Build > Firestore Database > Create database
  4. Start in test mode (or configure rules later)
  5. Go to Build > Authentication > Sign-in method > Enable Anonymous
  6. Go to Project settings (gear icon) > Your apps > Web app
  7. Copy the firebaseConfig object and paste below
